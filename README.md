# vir ~ ~ ~

## 环境准备

  >依赖 jQuery （ 1.x、2.x、3.x ）

  ```html
  <script src="https://unpkg.com/jquery"></script>
  ```

## 安装
### script 标签

  ```html
  <script src="https://unpkg.com/vir"></script>
  ```
### npm

  ```bash
  $ npm install vir -D
  # or
  $ yarn add vir -D
  ```
